下拉框组件

组件激活方法

当页面初始化的时候如果有下拉框的HTML代码，会自动激活下拉框。

页面中多数下拉框内容是需要动态加载的，可以配合angularJS,先加载出下拉列表数据然后激活下拉框。

手动激活下拉框JS部分

                        
                            $('#dropdownMenu').click(function () {
                                //可以在激活之前加载下拉列表数据
                                VA('#dropdownMenu').dropdown();
                            });

VA().dropdown()中可传入参数改变下拉框是向上弹出还是向下，默认向下。

VA().dropdown('up')向上弹出。

下拉框弹出后点击列表会自动显示到下拉框按钮中。

HTML代码

                        
                           <!-- dropDown -->
                           <div class="V_dropdown">
                               <button class="btn btn-default dropdown-toggle" type="button" id="dropdownMenu">
                                   Dropdown
                                   <span class="caret"></span>
                               </button>
                               <ul class="V_dropdown-menu" V-labelledby="#dropdownMenu">
                                   <li role="presentation">
                                       <span tabindex="-1">
                                           Action
                                       </span>
                                   </li>
                                   <li role="presentation">
                                       <span tabindex="-1">
                                           Another action
                                       </span>
                                   </li>
                                   <li role="presentation">
                                       <span tabindex="-1">
                                           Something else here
                                       </span>
                                   </li>
                                   <li role="presentation" class="divider"></li>
                                   <li role="presentation">
                                       <span tabindex="-1">
                                           Separated link
                                       </span>
                                   </li>
                               </ul>
                           </div>

如果要在下拉列表中加入分割线，直接添加<li class="divider"></li>。

回调函数

当用户点击下拉列表后，会返回出当前点击的数据或者文字。

                        
                            $('#dropdownMenu').click(function () {
                                 VA('#dropdownMenu').dropdown('down', function (event) {
                                     console.log(event);
                                  });
                             });

实例 ：

  • Action
  • Another action
  • Something else here
  • Separated link
